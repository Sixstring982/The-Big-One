The Big One --

Random room based dungeon crawler
Themed as a spy walking through a museum
Get treasure, avoid traps, use gadgets
to disarm traps.

Treasure gets randomly generated name

A "twist" at the end of each level, proceeding
to the next floor.

After all lives are used, the game is over.

Janitor's closet room contains items to
deactivate traps
Janitor is an NPC (Trader)

Trap ideas --

Pressure Plate - deactivated by deadweight.
Lasers - Spray can?
